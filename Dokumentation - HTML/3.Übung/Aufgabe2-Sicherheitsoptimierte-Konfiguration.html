Add-Ons: UBlock Origin, NoScript, HTTPS Everywhere, Privacy Badger installieren. Fuer Erklaerungen siehe Aufgabe 3.

Einstellungen:

  1. Allgemein
    • Immer überprüfen, ob Firefox der Standardrowser ist deaktivieren
    • Beim Start von Firefox, firefox eine leere Seite anzeigen lassen
    • Tabs erst laden, wenn sie ausgewaehlt werden aktivieren
  2. Suche
    • DuckDuckGo auswaehlen. DuckDuckGo verfolgt seine User bei der Suche, anders als z.B. Google, nicht
    • In Windows: Diese Suchmaschine für die Windows suche verwenden deaktivieren
    • Suchvorschläge amzeigen deaktivieren
    • Nicht verwendete Suchmachinen entfernen
  3. Inhalt
    • Pop-Ups blockieren aktivieren
  4. Anwedungen
    • Bei allen Anwendugen immer Nachfragren aktivieren
  5. Datenschutz
    • Unter "Tracking": Websiten auffordern, meine Aktivitäten nicht zu verfolgen auf true setzen. Dadurch wird an Websiten eine "do-not-track"-Aufforderung gesendet. Allerdings ist es den betroffenen Websiten selbst ueberlassen, dieser Aufforderung nachzukommen
    • Im selben Abschnitt die Checkbox "Tracking Protection in privaten Fenstern benutzen" auf true setzen. Erklaerung weiter unten unter "about:config"
    • Chronik nach benutzerdefinierten Einstellunge auswählen
      Hier Cookies akzeptieren auswählen, aber nicht von Drittanbietern und auch nur behalten, bis Firefox geschlossen wird
      Chronik Löschen wenn Firefox geschlossen wird aktiveren
    • Keine Vorschläge für die Adressleiste zulassen
  6. Sicherheit
    • Warnen, wenn Websiten versuchen Addons zu installieren aktivieren
    • "Webseite blockieren, wenn sie als attackierend gemeldet wurden" aktivieren
    • "Webseite blockieren, wenn sie als Betrugsversuch gemeldet wurde" aktivieren
    • "Zugansdaten für Websites speichern" und "Master-Passwortverwenden" deaktivieren
  7. Sync
    • Sich nicht mit einen Firefox-Konto anmelden
  8. Erweitert
    1. Allgemein
      • "Warnen, wenn Webseiten versuchen, umzuleiten oder neuzuladen" aktivieren
    2. Datenuebermittlung
      • Firefox-Absturzbericht und Absturzmelder auf deaktivieren
    3. Netzwerk
      • Autmoatisches Cache Management ausschalten und auf 500 MB begrenzen
      • "Nachfragen, wenn Websiten Daten fuer die Verwendung im Offline-Modus speichern moechten" auf aktivieren
    4. Update
      • Nach Updates Suchen, aber vor der Installation nachfragen auswählen
    5. Zertifikate
      • Jedes Mal nachfragen auswählen, wenn eine Webseite in persönliches Zertifikat verlangt
      • Aktuelle Gültigkeit von Zertifikaten durch Anfrage bei dem OCSP-Server bestätigen lassen


about:config:
trackingprotection.enabled => true. Hierdurch wird die eingebaute Tracking-Protection aktiviert, welche Domains blockt, die den User ueber mehrere Webseiten verfolgen.
geo.enabled => false. Hierdurch koennen Webseiten nicht auf die physische Position des Users, die u.U. auf Laptops gespeichert wird, zugreifen.
network.prefetch-next => false. Erklaerung siehe Aufgabe 1 Teilaufgabe j
network.http.pipelining => true
network.http.pipelining.ssl=> true
network.http.proxy.pipelining => true. Durch die drei vorangehenden Einstellungen wird http pipelining aktiviert, was dem Browser erlaubt, mehrere Objekte vom Server parallel zu laden, bspw. Bilder und Text gleichzeitig. Dies kann zwar bei alten oder schlecht konfigurierten Webseiten Probleme bereiten, stellt aber bei den meisten Webseiten eine Geschwindigkeitsverbesserung dar.
loop.enabled => false
browser.pocket.enable => false
media.navigator.enabled => false
media.peerconnection.enabled => false. Die drei vorhergehenden Einstellungen deaktivieren die Firefox-Applikationen Pocket, Hello und WebRTC. Diese werden von den meisten Usern nicht benoetigt und sollten daher deaktiviert werden.
