Web of Trust im Internet

  • Beim Versuch, auf das ZPA-System der FK07 verschlüsselt zuzugreifen erhält man den Hinweis, dass dem Aussteller des Zertifikates nicht standardäßig vertraut wird.
  • Dies kann möglicherweise daran liegen, dass die Domains der alten, nicht mehr gepflegten Seiten der HM "w3-o"" beinhalten.

  • Das Zertifikat kann in die Liste der vertraulichen Zertifikaten aufgenommen werden.

  • Nachdem dem Zertifikt nun vertraut wird kann das ZPA-System problemlos aufgerufen werden.

  • Beim aufrufen von www.bsi.bund.de erscheint jener Hinweis nicht. Die Seite wird also verschlüsselt aufgerufen.
  • Das liegt daran, dass dem Austeller des HTTPS-Zertikates von bsi.bund.de bereits standartmäßig vertraut wird.

  • Wenn ein HTTPS-Zertifikat von einer Quelle signiert wurde, der bereits (standardmäßig) vertraut wird, so wird auch dessen Zertifikaten vertraut.
    Damit wird es möglich mit einer Sicheren Verbundung zu surfen ohne erst des Zertifikat einer jeden Webseite selbst zu bestätigen.
